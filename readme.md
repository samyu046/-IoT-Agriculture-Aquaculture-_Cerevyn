# IoT – Agriculture & Aquaculture Smart Monitoring System

## Problem Statement
Design an IoT-based smart monitoring system for agriculture or aquaculture that monitors parameters and triggers alerts.

## Repository Contents

- **system_architecture.png**  
  Shows the overall IoT system architecture including sensors, edge gateway, cloud, and dashboard.

- **sensor_list.txt**  
  Contains the list of sensors used along with justification for each sensor.

- **data_flow.txt**  
  Explains how data flows from sensors to the cloud through the edge gateway.

- **sample_output.json**  
  Provides sample sensor data and alert output as displayed on the dashboard.

## Summary
This project demonstrates a practical IoT monitoring solution focusing on architecture design, data flow, and edge–cloud logic using simulated sensor data.
